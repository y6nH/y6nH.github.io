No Man's Science

In an aside to a previous post, I complained about the arbitrary chemistry of No Man’s Sky. Recently, I received an answer of sorts from the game itself.

Aboard the Anomaly, Nada the Robopriest is running the numbers.

“Physics simplistic. Chemistry arbitrary. Hypothesis: simulation.”

In other words, It’s only a game.
